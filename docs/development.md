# Development

## Setup

```bash
./scripts/setup.sh
```

## Run tests

```bash
./scripts/test.sh
```

## Run locally

```bash
./scripts/run.sh
```

The default local URL is <http://127.0.0.1:8080>.

## Build container

```bash
./scripts/build.sh
```

## Smoke checks

With the app running:

```bash
BASE_URL=http://127.0.0.1:8080 ./scripts/smoke.sh
```
